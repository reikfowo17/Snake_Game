# Snake Game

A simple classic Snake game written in C++ that runs directly in the Windows console.

## Controls
- `W`: Move Up
- `X`: Move Down
- `A`: Move Left
- `D`: Move Right

## How to run
1. Clone the repository:
   ```bash
   git clone https://github.com/reikfowo17/Snake_Game.git
   ```
2. Compile the game using a C++ compiler (like GCC/MinGW):
   ```bash
   g++ Snake.cpp -o SnakeGame.exe
   ```
3. Run the executable:
   ```bash
   ./SnakeGame.exe
   ```
